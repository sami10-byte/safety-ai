# موقع تقارير الصحة والسلامة المهنية بالذكاء الاصطناعي

نسخة أولية MVP لموقع عربي يولّد تقارير HSE من وصف المستخدم وصورة اختيارية.

## التشغيل المحلي

1. ثبّت Node.js.
2. افتح المجلد في الطرفية.
3. نفّذ:

```bash
npm install
cp .env.example .env
```

4. افتح ملف `.env` وضع مفتاح OpenAI:

```bash
OPENAI_API_KEY=your_api_key_here
```

5. شغّل الموقع:

```bash
npm start
```

ثم افتح:

```text
http://localhost:3000
```

## ملاحظات مهمة

- المفتاح يجب أن يبقى في السيرفر ولا يوضع داخل ملفات الواجهة.
- التقرير الناتج يحتاج مراجعة مسؤول السلامة قبل الاعتماد.
- يمكن رفعه لاحقًا على Render أو Railway أو VPS.
